# Tags

Building blocks of HTML pages. Used to define the structure and content of a web page.

```html
<html>
  <head>
    <!-- meta data information here-->
    <title>Title of the Page</title>
  </head>
  <body>
    <!-- "meat" of the page thatre rendered here -->
    <!-- e.g. text buttons -->
    <div>div is a block element</div>
    <span>Span is an in-line element</span>
    <span>!!!</a>
    <!-- span is in-line, thus takes up only as much space as needed -->
    <div>hi</div>
  </body>
</html>
```

# Attributes

Provides additional information about HTML elements.

Common ones:

- `class`: specifies one or more classnames for an element, usually used with CSS.

- `id`: specifies a unique id for an element.

- `src`: used in `<img>` tags to define the URL of an image.

- `href`: used in `<a>` tags to define the hyperlink - destination.

- `style`: contains CSS styling for the element.
