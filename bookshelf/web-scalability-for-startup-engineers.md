## Web Scalability for Startup Engineers by Artur Ejsmont

<img src="/photos/books/web-scalability-for-startup-engineers.jpeg" height ="300">

This book is a lower level version of the book, Designing Data Intensive Applications (DDIA) as that it talks about the implementation of some technologies in DDIA, as well as design patterns and principles as well as software architecture. It is much more focused on web technologies (as its title would suggest) than DDIA, and is an easier read.

<img src="/photos/books/page/web-scalability-for-startup-engineers-book.jpeg">

I'll come to update this page once I've finished reading DDIA, which may take a long while. I havent read this book from front to cover yet. I've only read from random sections to compare with the content from the book DDIA. Thus, I could say it is a good companion book to DDIA. Whenever I feel I need to learn more from DDIA, I would look for a relevant section from Web Scalability for Startup Engineers and read it there, then I'd google to learn more.
