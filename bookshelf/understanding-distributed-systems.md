## Understanding Distributed Systems

<img src="https://m.media-amazon.com/images/I/71rlI0H7QsL._SY522_.jpg" height="300">

This is a good book. I read this as I wanted an easier read to the extremely dense book, Designing Data Intensive Applications (DDIA). But after reading this book, I didnt find the time sunk to have been worth my time. It was extremely basic. The reason I still find it a good book, is that it's well written for an audience that's just looking for a shallow covering of distributed system concepts, like perhaps a freshman student. But for someone with a bit of experience with the web, sticking to books like Designing Data Intensive Applications (DDIA) or Web Scalability for Startup Engineers wouldve been a far better use of one's time.
